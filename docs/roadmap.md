## 6. Definition of Done (DoD) для MVP

### На уровне User Story:
- [ ] Код написан и прошел code review (2 аппрувера)
- [ ] Unit-тесты покрывают бизнес-логику (>80% coverage для сервисов)
- [ ] Интеграционные тесты для API-эндпоинтов
- [ ] UI соответствует дизайн-макетам (pixel-perfect не требуется, но UX -- да)
- [ ] Responsive: работает на desktop (1280+), tablet (768+), mobile (375+)
- [ ] WCAG 2.1 AA контрастность для текста
- [ ] Данные валидируются на фронте и бэке (Zod-схемы)
- [ ] Нет критических и высоких уязвимостей (OWASP Top 10 проверка)
- [ ] Задокументированы API-эндпоинты (OpenAPI авто-генерация)
- [ ] Задача прошла QA на staging-среде
- [ ] Product Owner подтвердил приемку

### На уровне MVP Release:
- [ ] Все Must-требования из PRD реализованы и приняты
- [ ] Нет открытых блокеров (P0) и критических багов (P1)
- [ ] P2-баги задокументированы, есть план исправления
- [ ] Load-тест пройден: 500 concurrent users, 50 AI-сессий, API p95 < 500ms
- [ ] Security-аудит пройден (OWASP, ФЗ-152 чеклист)
- [ ] Мониторинг и алерты настроены (Grafana, Sentry)
- [ ] Backup и восстановление БД протестированы
- [ ] Юридические документы готовы (оферта, политика конфиденциальности, согласие на ПД)
- [ ] 50+ специалистов зарегистрированы и прошли ИИ-интервью
- [ ] Инструкция по эксплуатации для команды поддержки
- [ ] Runbook для инцидентов

## 9. Ключевые решения и допущения

### Решения:
1. MVP -- веб-приложение (responsive), без нативных мобильных приложений
2. Модульный монолит NestJS, не микросервисы
3. Claude 3.5 Sonnet как основная LLM, GPT-4o-mini как fallback
4. pgvector в PostgreSQL, не отдельный vector DB
5. Спринты по 2 недели, Scrum-фреймворк
6. Monorepo (Turborepo) для фронта, бэка и общих пакетов

### Допущения:
1. Команда из 8-10 человек будет полностью укомплектована к Sprint 0
2. Anthropic API будет доступен из РФ (через прокси при необходимости)
3. ЮKassa KYC-процесс займет не более 3-4 недель
4. К моменту бета-запуска будет набрано минимум 50 специалистов
5. LLM-модели достаточного качества для проведения ценностного интервью
6. Бюджет на инфраструктуру и LLM утвержден
