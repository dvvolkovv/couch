### 1.2. Выводы из конкурентного анализа цен

1. **Стандарт рынка -- комиссия 20-30%.** Все основные российские конкуренты берут комиссию с каждой сессии. Это создает проблему: специалисты мотивированы уводить клиентов с платформы.

2. **B17.ru -- единственный с подпиской для специалистов.** Но это "доска объявлений", а не маркетплейс с добавленной ценностью. Подписка работает при генерации лидов.

3. **Ни одна российская платформа не берет деньги с клиентов** за подписку/сервис (только за сессии). Это возможность для дифференциации, но и риск.

4. **Ценовой диапазон сессий: 1 500 - 6 000 руб.** Медиана ~3 500 руб. SoulMate не должна устанавливать цены -- это делают специалисты.

5. **Пакеты сессий** предлагает Zigmund -- это работающая механика повышения retention.

### 1.3. Расчет экономики специалиста на конкурирующих платформах

**Специалист со средней нагрузкой (20 сессий/мес через платформу, цена 3 500 руб.):**

| Платформа | GMV | Комиссия | Подписка | Чистый доход |
|-----------|-----|----------|----------|-------------|
| Ясно (25%) | 70 000 | -17 500 | 0 | 52 500 |
| Alter (22%) | 70 000 | -15 400 | 0 | 54 600 |
| B17.ru | 70 000 | 0 | -5 000 | 65 000 |
| **SoulMate (Старт)** | 70 000 | -14 000 (20%) | 0 | 56 000* |
| **SoulMate (Проф.)** | 70 000 | -8 400 (12%) | -2 990 | 58 610 |
| **SoulMate (Эксп.)** | 70 000 | -5 600 (8%) | -5 990 | 58 410 |

*Лимит 5 клиентов/мес ограничивает число сессий на тарифе "Старт".

**Вывод:** Тариф "Профессионал" SoulMate дает специалисту чистый доход на уровне лучших конкурентов, а при росте объема -- значительно выше. Порог перехода на "Профессионал" -- от 10-12 сессий/мес через платформу.

## 2. Принципы ценообразования SoulMate

### 2.1. Стратегические принципы

1. **Нулевой барьер входа.** Как специалист, так и клиент могут начать работу бесплатно. Это критично для cold start проблемы маркетплейса.

2. **Ценность определяет цену.** Платные тарифы привязаны к конкретной бизнес-ценности: больше клиентов, ниже комиссия, расширенные инструменты.

3. **Прогрессивная монетизация.** По мере роста использования платформы пользователь естественно "вырастает" из бесплатного тарифа.

4. **Прозрачность.** Никаких скрытых комиссий. Специалист видит, сколько платит. Клиент видит финальную цену.

5. **Конкурентоспособность.** Эффективная комиссия для активных специалистов ниже, чем у Ясно/Alter.

### 2.2. Модель: Freemium + Hybrid Commission

Гибридная модель сочетает три потока:

```
+----------------------------------+
|  ПОТОК 1: Комиссия с сессий      |
|  8-20% в зависимости от тарифа   |
|  ~70% выручки Год 1               |
+----------------------------------+
           +
+----------------------------------+
|  ПОТОК 2: Подписки специалистов  |
|  2 990 / 5 990 руб./мес          |
|  ~25% выручки Год 1               |
+----------------------------------+
           +
+----------------------------------+
|  ПОТОК 3: Premium-подписка       |
|  499 руб./мес для клиентов        |
|  ~5% выручки Год 1                |
+----------------------------------+
```

## 3. Тарифы для специалистов

### 3.1. Тариф "Старт" (бесплатный)

| Параметр | Значение |
|----------|----------|
| **Цена** | 0 руб./мес |
| **Комиссия** | 20% от стоимости каждой сессии |
| **Лимит клиентов** | До 5 новых клиентов через платформу в месяц |
| **Лимит сессий** | Неограниченно (с имеющимися клиентами) |
| **ИИ-интервью** | Включено |
| **Профиль в каталоге** | Базовый (фото, описание, специализация, цена) |
| **Аналитика** | Базовая (число просмотров, бронирований) |
| **Приоритет в выдаче** | Стандартный |
| **Видео-визитка** | Нет |
| **Бейдж** | Нет |

**Обоснование цены:** Нулевой барьер входа решает проблему cold start. Комиссия 20% -- на уровне рынка (Ясно 20-30%). Лимит 5 клиентов/мес создает естественное давление на переход к платному тарифу при росте потока.

**Экономика для специалиста:**
- 5 клиентов x 4 сессии/мес = 20 сессий
- GMV: 20 x 3 500 = 70 000 руб.
- Комиссия: 14 000 руб.
- Чистый доход: 56 000 руб.

|----------|----------|

| **Комиссия** | 12% от стоимости каждой сессии |
| **Лимит клиентов** | Без ограничений |
| **ИИ-интервью** | Включено + обновление 1 раз/квартал |

- Подписка: 2 990 руб.
- Итого расходы: 9 290 руб.
- **Эффективная комиссия: 17.7%** (vs 20% на "Старт")

**Экономика для специалиста (25 сессий/мес):**

## 4. Тарифы для клиентов

### 4.1. Бесплатный тариф

| Параметр | Значение |
|----------|----------|
| **Цена** | 0 руб. |
| **ИИ-консультация** | 1 полная консультация (повторная через 30 дней) |
| **Каталог специалистов** | Полный доступ |
| **Рекомендации** | Top-5 специалистов (1 раз после ИИ-консультации) |
| **Бронирование** | Без ограничений |
| **Трекер прогресса** | Нет (Phase 2) |

**Обоснование:** Основная ценность для клиента -- подбор специалиста. Первая ИИ-консультация бесплатна всегда -- это ключевая воронка привлечения. Повторная консультация через 30 дней -- если клиент хочет сменить специалиста.

### 4.2. Тариф "Premium"

| Параметр | Значение |
|----------|----------|
| **Цена** | 499 руб./мес |
| **ИИ-консультация** | Неограниченные (обновление профиля, пересмотр рекомендаций) |
| **Рекомендации** | Расширенные (Top-10, детальные объяснения, фильтры) |
| **Приоритетное бронирование** | Раннее уведомление о свободных слотах |
| **Трекер прогресса** | Да (Phase 2, записи заметок после сессий, графики) |
| **Скидка на пакеты** | Дополнительная скидка 5% на пакеты сессий |
| **Поддержка** | Приоритетная |

**Обоснование цены:**
- 499 руб. = ~14% от средней стоимости одной сессии (3 500 руб.)
- Для клиента, посещающего 4 сессии/мес (14 000 руб.) -- это 3.6% от расходов на терапию
- Психологический порог "меньше 500 рублей" -- импульсная покупка
- Основная ценность -- неограниченные ИИ-консультации (для тех, кто ищет/меняет специалиста)

### 4.3. Когда Premium не имеет смысла

- Клиент нашел специалиста с первого раза и не планирует менять
- Клиент ходит на сессии реже 1 раза в месяц

Это ограничивает конверсию в Premium до 5-8% базы -- реалистичная оценка для freemium.

## 5. Комиссионная структура

### 5.1. Как работает комиссия

```
Клиент оплачивает: 3 500 руб. (полная стоимость сессии)
        |
        v
+------------------+
|   ЮKassa         |
|   (-3.5% = 122)  |
+------------------+
        |
        v
   SoulMate получает: 3 378 руб.
        |
        +--- Комиссия SoulMate (зависит от тарифа спец.):
        |    Старт: 20% = 700 руб. (из 3 500)
        |    Проф.: 12% = 420 руб.
        |    Эксп.: 8%  = 280 руб.
        |
        +--- Выплата специалисту:
             Старт: 2 800 руб.
             Проф.: 3 080 руб.
             Эксп.: 3 220 руб.
```

**Фактическая чистая маржа SoulMate с сессии (после эквайринга):**

| Тариф специалиста | Комиссия | Эквайринг (3.5%) | Чистый доход |
|-------------------|----------|-------------------|-------------|
| Старт (20%) | 700 руб. | 122 руб. | 578 руб. |
| Профессионал (12%) | 420 руб. | 122 руб. | 298 руб. |
| Эксперт (8%) | 280 руб. | 122 руб. | 158 руб. |

**Важно:** Эквайринг взимается со всей суммы транзакции. При комиссии 8% (Эксперт) эквайринг съедает 44% комиссии. Это компенсируется фиксированной подпиской.

### 5.2. Пакеты сессий

Для тарифов "Профессионал" и "Эксперт" доступно создание пакетов:

| Пакет | Скидка для клиента | Комиссия SoulMate |
|-------|--------------------|-------------------|
| 4 сессии | 5% | Такая же (12% или 8%) |
| 8 сессий | 10% | Такая же |
| 12 сессий | 15% | Такая же |

**Финансовый эффект пакетов:**
- Снижение GMV на 5-15% за счет скидки
- Увеличение retention на 20-30% (клиент "привязан" к пакету)
- Снижение churn -> рост LTV -> нетто-позитивный эффект

Пример: Пакет 8 сессий x 3 500 x 0.9 = 25 200 руб. (вместо 28 000)
- Комиссия SoulMate (12%): 3 024 руб. (вместо 3 360)
- Потеря: 336 руб.
- Но retention пакета ~85% vs ~65% без пакета -> +2.8 дополнительных сессии в среднем

### 5.3. Политика выплат специалистам

| Параметр | Значение |
|----------|----------|
| Частота выплат | 2 раза в месяц (1-го и 16-го) |
| Минимальная сумма выплаты | 1 000 руб. |
| Способы выплаты | Банковская карта, банковский счет (через ЮKassa Payouts) |
| Комиссия за выплату | За счет SoulMate |
| Холд средств | 48 часов после завершения сессии (защита от возвратов) |

## 7. Анализ ценовой чувствительности

### 7.1. Чувствительность подписки специалистов

**Вопрос:** При какой цене подписки "Профессионал" максимизируется выручка?

| Цена | Ожид. конверсия | Доход от подписок (при 530 спец.) | Потеря комиссий | Нетто-эффект |
|------|----------------|----------------------------------|----------------|-------------|
| 1 990 руб. | 40% | 424 000 | -106 000 (ниже комиссия) | +318 000 |
| **2 990 руб.** | **30%** | **476 000** | **-79 000** | **+397 000** |
| 3 990 руб. | 22% | 466 000 | -58 000 | +408 000 |
| 4 990 руб. | 15% | 397 000 | -40 000 | +357 000 |

**Вывод:** Оптимум находится в диапазоне 2 990 - 3 990 руб. Рекомендуется начать с 2 990 руб. (ниже барьер входа) и тестировать 3 490 руб. через A/B тест после 6 месяцев.

### 7.2. Чувствительность Premium-подписки клиентов

| Цена | Ожид. конверсия | MRR от Premium (при 3 100 клиентах) | Каннибализация |
|------|----------------|-------------------------------------|---------------|
| 299 руб. | 10% | 93 000 | Минимальная |
| **499 руб.** | **6.5%** | **100 000** | Минимальная |
| 699 руб. | 4% | 87 000 | Есть (клиенты ожидают скидку на сессии) |
| 999 руб. | 2% | 62 000 | Значительная |

**Вывод:** 499 руб./мес -- оптимальная точка. При 299 руб. конверсия не растет пропорционально снижению цены. При 699+ руб. конверсия резко падает, т.к. клиенты сравнивают с ценой сессии.

### 7.3. Чувствительность к уровню комиссии

**Сценарий: Конкурент (Ясно) снижает комиссию до 15%:**

| Наш ответ | Влияние на выручку | Влияние на базу спец. |
|-----------|-------------------|----------------------|
| Оставить как есть | -10% (отток спец.) | Потеря 15-20% спец. |
| Снизить "Старт" до 15% | -25% комиссий | Удержание базы |
| Снизить "Проф." до 10% | -17% комиссий, +5% подписок | Привлечение спец. |
| Запустить промо-период | -5% (временно) | Временное удержание |

**Рекомендация:** Основное конкурентное преимущество -- качество матчинга, а не цена. При ценовом давлении рекомендуется снижать комиссию на тарифе "Старт" (до 15%), сохраняя преимущество платных тарифов.

## 8. План A/B тестирования цен

### 8.1. Приоритеты тестирования

| Приоритет | Тест | Гипотеза | Метрика | Срок |
|-----------|------|----------|---------|------|
| 1 | Цена "Профессионал": 2 990 vs 3 490 | 3 490 не снижает конверсию >10% | Конверсия в подписку, churn | Мес 4-6 |
| 2 | Цена Premium: 499 vs 399 vs 599 | 499 -- оптимум | Конверсия, revenue per user | Мес 5-7 |
| 3 | Комиссия "Старт": 20% vs 18% vs 15% | 15% увеличивает привлечение на >20% | Рост базы спец., retention | Мес 6-8 |
| 4 | Пакеты: скидка 10% vs 15% | 15% не снижает revenue vs retention gain | Retention, LTV, revenue | Мес 7-9 |
| 5 | Годовая подписка: 17% vs 25% скидка | 25% увеличивает конверсию в годовую на >50% | % годовых подписок, churn | Мес 8-10 |

### 8.2. Методология

**Сегментация:** По region_id (город) или по user_id % N для равномерного распределения.

**Минимальная выборка:** 200 пользователей на вариант для подписок специалистов, 500 для клиентских тестов.

**Длительность:** Минимум 4 недели (полный цикл оплаты подписки).

**Критерий успеха:** Статистическая значимость p < 0.05. Метрика -- Revenue per User (не просто конверсия).

### 8.3. Тесты, которые нельзя проводить

- Разная комиссия для одной категории специалистов в одном регионе (жалобы, "несправедливость")
- Разные цены сессий (цены устанавливает специалист)
- Тест Premium vs бесплатных функций (нарушение ожиданий)

| Английский рынок (глобальный) | 24+ мес | USD | Полная переработка тарифной сетки |

### 9.2. Принципы адаптации цен

1. **Паритет покупательной способности (PPP).** Цены адаптируются по индексу PPP Всемирного банка.
2. **Конкурентная среда.** В каждом рынке -- анализ локальных конкурентов.
3. **Единая валюта учета.** Все метрики пересчитываются в RUB для отчетности.
4. **Специалисты устанавливают цены в локальной валюте.** Платформа не конвертирует и не арбитражит.

## Приложение: Калькулятор "Переход на платный тариф"

### Для специалиста: "Когда выгодно перейти на Профессионал?"

```
Если вы проводите через SoulMate N сессий в месяц по цене P руб.:

Тариф "Старт":
  Расходы = N x P x 0.20

Тариф "Профессионал":
  Расходы = 2 990 + N x P x 0.12

Точка безразличия:
  N x P x 0.20 = 2 990 + N x P x 0.12
  N x P x 0.08 = 2 990
  N x P = 37 375

При P = 3 500: N = 10.7 сессий
При P = 2 500: N = 14.9 сессий
При P = 5 000: N = 7.5 сессий
```

### Для специалиста: "Когда выгодно перейти на Эксперт?"

```
Тариф "Профессионал":
  Расходы = 2 990 + N x P x 0.12

Тариф "Эксперт":
  Расходы = 5 990 + N x P x 0.08

Точка безразличия:
  2 990 + N x P x 0.12 = 5 990 + N x P x 0.08
  N x P x 0.04 = 3 000
  N x P = 75 000

При P = 3 500: N = 21.4 сессий
При P = 2 500: N = 30.0 сессий
При P = 5 000: N = 15.0 сессий
```
