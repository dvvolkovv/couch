## ЧАСТЬ 4: АДАПТИВНЫЕ ЗАМЕТКИ

### Общие правила адаптивности

```
Desktop (>= 1024px):
  - Sidebar + Content layout для дашбордов
  - 2-3 колонки для карточек каталога
  - Модальные окна по центру экрана
  - Фильтры -- левая колонка (sticky)
  - Чат ИИ: 65% чат + 35% прогресс

Tablet (768px - 1023px):
  - Content занимает всю ширину
  - 2 колонки для карточек
  - Фильтры -- collapsible header или drawer
  - Чат ИИ: полная ширина, прогресс в header

Mobile (< 768px):
  - 1 колонка
  - Bottom navigation (5 вкладок)
  - Bottom sheets вместо модалок
  - Фильтры -- bottom sheet с handle
  - Чат ИИ: fullscreen, прогресс по кнопке
  - Все кнопки: full-width, min-height 48px
  - Sticky footer для ключевых CTA
```
